The first week of my adventure came to an end last Sunday, and I must say it was an enjoyable venture.

Metaphorically I can probably sum up my week with the following quotes:

 

"With mirth and laughter let old wrinkles come."
-- William Shakespeare

Joy. Working with the folks in A*Team has been an absolute pleasure. They're fun, clever and they work really hard.

 

"A fool thinks himself to be wise, but a wise man knows himself to be a fool."
-- William Shakespeare

Python. Every time I begin to feel that I have obtained a certain degree of proficiency with the language, the great snake returns to remind me that I still have much to learn.

 

"There are more things that vi(m) can do, Horatio, than are dreams in your philosophy."
-- vi folklore

Vim. My tryst with this magical editor continues, and with each passing day, I hope to unearth a new secret.

 

Coming back to the technical side of things, Over the last week I've been working on writing tests for two python modules titled "mozfile" and "mozinfo".

  1. Mozfile, is a module that lets one handle simple file operations without worrying about the underlying platform inconsistencies.You can track progress for this on  Bug 885224.
  2. Mozinfo, is a module that gives you platform information about the operating system from inside Python in a convenient form.You can track progress for this on Bug 885145.


 

Coming up next are Moznetwork and Mozinstall, so keep a eye out for updates!
